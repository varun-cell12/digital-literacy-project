### 🔹 Task 1 – Infographic

Created a digital literacy awareness infographic using Canva. It covers topics such as digital literacy, useful tools, safe internet practices, and professional online presence.

---

### 🔹 Task 2 – Digital Portfolio

Developed profiles on GitHub, LinkedIn, and HackerRank. These platforms help in building a professional digital presence and improving technical skills.
